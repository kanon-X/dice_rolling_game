# 🎲 Python Dice Rolling Game

A simple command-line dice rolling game built using Python.  
This project is great for beginners to understand loops, conditionals, and random number generates.

---

## 🚀 Features

- Roll two dice 🎲🎲
- Random number generation (1–6)
- Continuous gameplay using a loop
- User input handling
- Input validation

---

## 🧠 How It Works

- The program asks the user whether they want to roll the dice.
- If the user enters **'y'**, two random numbers (1–6) are generated.
- If the user enters **'n'**, the game exits.
- Any other input is treated as invalid.

---

## 🛠️ Technologies Used

- Python 3
- `random` module

---

1. Make sure Python is installed on your system.
2. Clone this repository or download the code.
